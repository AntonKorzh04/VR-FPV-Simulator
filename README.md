# VR-FPV-Simulator
Симулятор управления FPV дроном (БПЛА)
Идея: управление FPV-дроном/БПЛА самолетного типа, поиск и поражение вражеской техники различными методами.
Ключевые механики:
•	управление скоростью, креном/тангажом с помощью контроллеров,
•	физика полета и столкновений
•	механика уничтожений транспорта врага,
•	видеопоток от первого лица.
Самая важная механика: реалистичная физика полета.

MVP: управление FPV дроном от первого лица с реалистичной физикой.
